# 🚖 Uber Power BI Dashboard

An interactive Business Intelligence dashboard built using **Power BI** to analyze Uber ride data and uncover actionable insights related to bookings, revenue, customer experience, and operational performance.

---

## 📊 Dashboard Overview

This dashboard provides a comprehensive view of Uber ride operations through dynamic visualizations and KPI tracking.

### Key Metrics
- ✅ Total Completed Bookings
- ❌ Lost Bookings
- 💰 Total Revenue
- 📍 Total Distance Covered
- 📏 Average Trip Distance
- ⭐ Customer Ratings
- 🚗 Driver Ratings

---

## 🔍 Features

### Booking Analysis
- Track completed and lost bookings
- Analyze monthly booking trends
- Monitor ride demand patterns

### Revenue Insights
- Revenue performance over time
- Vehicle-wise revenue contribution
- Business growth monitoring

### Location Analysis
- Top Pickup Locations
- Top Drop Locations
- Location-based demand insights

### Customer Experience
- Average customer ratings
- Average driver ratings
- Service quality monitoring

### Vehicle Performance
- Compare vehicle categories
- Analyze revenue generated by each vehicle type
- Identify high-performing ride segments

---

## 🛠️ Tools & Technologies

- **Power BI Desktop**

- **Data Modeling**
